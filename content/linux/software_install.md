notepadqq:

Ubuntu下的安装方法:

	sudo add-apt-repository ppa:notepadqq-team/notepadqq

	sudo apt-get update

	sudo apt-get install notepadqq

Ubuntu下的卸载方法:

	sudo apt-get remove notepadqq

s	udo add-apt-repository --remove ppa:notepadqq-team/notepadqq
